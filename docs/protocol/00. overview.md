# 00. Protocol Overview & Quick Search Guide

## Core Protocol Architecture

The MouseWithoutBorders protocol is designed for extreme low-latency and lightweight communication across a local network. It achieves this by bypassing typical application-layer serialization (like JSON or XML) and instead communicating via fixed-size C-style structs mapped directly to byte arrays. 

This document serves as the master index and quick search guide. Use the keywords to quickly locate the concepts you need to implement.
